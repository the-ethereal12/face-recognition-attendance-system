# Face Recognition Attendance System

## Overview
A real-time Face Recognition–based Attendance System built using Python, OpenCV, and the LBPH (Local Binary Pattern Histogram) algorithm.  
The system detects and recognizes faces through a webcam feed and automatically records attendance in a CSV file.

---

## Features
- Face detection using Haar Cascade classifier
- Face recognition using LBPH algorithm
- Dataset generation for new users
- Model training and label encoding
- Automatic attendance marking in CSV
- Flask-based web interface
- Real-time webcam integration

---

## Tech Stack
- Python
- OpenCV
- LBPH Face Recognizer
- Haar Cascade Classifier
- Flask
- NumPy
- Pickle
